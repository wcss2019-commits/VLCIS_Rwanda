# VLCIS Rwanda 🌍

AI-powered Climate Risk & Early Warning System for Rwanda.

## Features
- Rainfall forecasting
- Flood risk modeling
- Real-time alerts
- API (FastAPI)

## Project Structure
- app/ → core system (API, services)
- data/ → input datasets
- outputs/ → results and alerts
- run_system.py → main pipeline
